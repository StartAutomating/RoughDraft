Extension/ListProtocols.rd.ext.ps1
----------------------------------

### Synopsis
Lists protocols

---

### Description

Lists the protocols available to ffmpeg.

---

### Examples
> EXAMPLE 1

```PowerShell
Get-FFMpeg -ListProtocol
```

---

### Parameters
#### **Protocol**

|Type      |Required|Position|PipelineInput|Aliases                    |
|----------|--------|--------|-------------|---------------------------|
|`[Switch]`|true    |named   |false        |ListProtocol<br/>Protcolols|
